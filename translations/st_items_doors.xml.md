## st_placeable_door

### EN:
```
Wooden Door 100x190cm
```

### RU:
```
Деревянная дверь 100x190см
```
## st_placeable_door_descr

### EN:
```
%c[ui_gray_3]PROPERTIES:\n
		%c[d_orange] • %c[ui_gray_2] placeable item\n
		%c[d_orange] • %c[ui_gray_2] Compatible with the craftable doorways\n
		%c[0,250,250,0] • %c[0,250,250,0] NOTICE: To remove the door stab it with a melee weapon or use explosive. Shooting it works too but requires a lot of bullets (for the purpose of preventing NPCs from breaking doors easly when shooting).\n
```

### RU:
```
%c[ui_gray_3]ОСОБЕННОСТИ:\n
		%c[d_orange] • %c[ui_gray_2] Размещаемый предмет\n
		%c[d_orange] • %c[ui_gray_2] Совместимость с дверными проемами\n
		%c[0,250,250,0] • %c[0,250,250,0] ВНИМАНИЕ: Чтобы выбить дверь, проткните ее холодным оружием или используйте взрывчатку. Стрельба тоже работает, но требует много пуль (чтобы NPC не могли легко выламывать двери при стрельбе).\n
```
