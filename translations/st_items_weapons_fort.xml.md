## st_wpn_fort12_descr

### EN:
```
The Fort-12 is an attempt for recently-independent Ukraine (from August 1991) to create its own police pistol to replace the Soviet-era Makarov PM, while still using the same 9x18 caliber. Designed in 1995 and developed in 1998 by Ukrainian firearms designer RPC Fort, this is a formidable pistol considered to be an upgrade of the PM itself. Although it may have reliability issues, especially due to the lack of the safe decocking facility, it compensates for the accuracy, magazine and ergonomics. This model comes with a threaded barrel, allowing it to equip silencers by default. \n \n

```
The Fort-17 is an improvement of the Fort-12 by replacing the frame from steel to polymer, which decreases weight, although the recoil might be just a little bit worse. Comes with magazines size 13 by default instead of 12 or 13. It also has a removable grip backstrap, which can improve ergonomics. \n \n
